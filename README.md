# 漫画ビューワ

モダンで高機能な漫画ビューワーアプリケーション

## 機能

- 📖 単ページ/見開き表示切替
- 🎨 ダークモード対応
- 🖱️ マウスホイールでズーム
- 📱 タッチ操作対応（スワイプ、ピンチズーム）
- 🔖 自動しおり機能
- ⚙️ カスタマイズ可能な設定
- 📊 サムネイル一覧
- 🎞️ 自動ページめくり
- ⌨️ キーボードショートカット対応

## キーボード操作

- `←` / `Space`: 次のページ（日本式）
- `→`: 前のページ（日本式）
- `S`: 表示モード切替
- `F`: 全画面表示
- `Home`: 最初のページへ
- `End`: 最後のページへ

## 使い方

1. ブラウザで `index.html` を開く
2. 画面をクリックまたはキーボードでページ移動
3. マウスホイールでズーム
4. 設定ボタンからカスタマイズ可能

## 技術スタック

- Pure HTML5
- Vanilla JavaScript (ES6+)
- CSS3 with Modern Features
- LocalStorage for Settings & Bookmarks
